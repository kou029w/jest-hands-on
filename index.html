Jest ハンズオン

CodeSandboxの場合、[Tests]のパネルを開いて、テスト結果を確認します。

ローカル環境の場合、 npm test コマンドを実行して、テスト結果を確認します。

ドキュメント

リファレンスとして Jestのドキュメント を使用します。ご参照ください。

構成

src/*.test.js がテスト本体です。 ファイルを配置することによって実行可能です。

  • はじめましょう … sum.js, sum.test.js
  • Matcherを使用する … toBe.test.js, toEqual.test.js, notToBe.test.js
  • 非同期コードのテスト … callback.test.js, promise.test.js
  • セットアップと破棄 … setupTeardown.test.js, scope.test.js
  • モック関数 … mock.test.js
